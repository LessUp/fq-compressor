## Reproduction

```bash
./scripts/build.sh clang-release
FQC_PERF_SIZES=64 FQC_PERF_REPEATS=3 FQC_PERF_DATA=random \
  FQC_PERF_MEMORY_MIB=64 FQC_PERF_ENFORCE_SLA=1 \
  FQC_PERF_KEEP_TEMP=1 bash tests/e2e/test_performance.sh
```

## Caveats

- WSL2 wall-clock noise is high; re-runs on the same host have observed Illumina compress as low
  as 47.8 MiB/s. The 53.15 figure is a best-case median, not a stable guarantee.
- ARM64 is unverified.
- Real biological corpora are not claimed; codec admission is governed by
  `benchmark_v2/CODEC_GATES.md`.
- LeakSanitizer and the ASan vptr check are unavailable in this environment; see
  `troubleshooting/0006-sanitizer-env-limitations.md`.
