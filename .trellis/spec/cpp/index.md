# C++ Development Guidelines

> Best practices for C++ development in this project.

---

## Overview

This directory contains guidelines for C++ development in fq-compressor, a high-performance FASTQ compression tool using C++23.

---

| Guide | Description |
|-------|-------------|

| [Testing Guidelines](./testing-guidelines.md) | GTest patterns, property-based testing |

---

## Language Requirements

- **Standard**: C++23
- **Compilers**: GCC 14+ or Clang 18+
- **Build System**: CMake 3.28+ with Ninja
- **Package Manager**: Conan 2.x

---

**Language**: All documentation should be written in **English**.
