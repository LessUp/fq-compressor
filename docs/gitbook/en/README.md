# fq-compressor Documentation

fq-compressor is a high-performance FASTQ compression project focused on:

- domain-aware compression for sequencing data
- block-based random access
- oneTBB parallel pipelines
- modern C++ engineering practices
- differentiated strategies for short and long reads
