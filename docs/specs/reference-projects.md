# 参考项目摘要

本文件用于承接原先分散在 `ref-projects/` 与规格目录中的参考项目结论。

## 参考项目列表

### fastq-tools

- 提供现代 C++ FASTQ 工具工程化参考
- 对脚本组织、代码风格和流水线结构有较大启发

### fqzcomp5

- 为质量值压缩提供上下文建模思路
- 对范围编码、统计压缩方向有重要参考价值

### Spring

- 是短读 ABC 路线的重要核心参考
- 对重排、局部共识、delta 编码影响很大
- 但原生设计存在短读长度限制

### Mincom

- 在 minimizer bucketing 方向提供了重要启发

### repaq

- 对压缩比与工程实现的平衡有参考意义

### pigz

- 对并行压缩架构和 Producer-Consumer 设计有参考价值

### NanoSpring

- 适合作为长读 overlap-based 路线对照
- 也帮助说明为何长读不宜直接套用短读 ABC

### HARC / minicom

- 提供更偏算法实验方向的序列压缩参考

## 对当前项目的建议

- 把参考项目的**结论**保留在文档里
- 不再把参考项目源码副本作为正式仓库结构的一部分
- 真正需要集成的外部代码，应进入 `vendor/` 并配套说明许可与构建方式
