# fq-compressor

**fq-compressor** is a high-performance FASTQ compressor built in **C++23** with
**Assembly-based Compression (ABC)**, **Statistical Context Mixing (SCM)**, and
**O(1) random access**.

## Why this project matters

- **Storage reduction** for sequencing datasets without giving up future access.
- **Random access** so downstream workflows can extract subsets without full decompression.
- **Modern implementation** using CMake presets, Conan, clangd, GitHub Actions, and OpenSpec.
