# fq-compressor

**High-performance FASTQ compression for the sequencing era**

A C++23 FASTQ compressor with **3.97× compression** and **O(1) random access**. Built for production FASTQ archives.

## Key Metrics

| Metric | Value |
|--------|-------|
| **Compression Ratio** | 3.97× on Illumina data |
| **Compression Speed** | 11.9 MB/s (multithreaded) |
| **Decompression Speed** | 62.3 MB/s (multithreaded) |
| **Random Access** | O(1) without full decompression |

## Why fq-compressor

1. **Storage reduction with retrieval** — Compress aggressively without sacrificing random access or verification
2. **Fast to practical use** — Prebuilt binaries, simple CLI, documented defaults
3. **Modern C++23 stack** — Intel oneTBB, CMake presets, Conan 2, clangd-friendly
4. **Archive inspection** — Block-based access, metadata queries, checksums
5. **Transparent input handling** — Works with .gz, .bz2, .xz inputs transparently
6. **Clean engineering workflow** — OpenSpec-driven repo, enforced quality gates, archive-grade maintenance

## Quick Start

<Tabs items={['Install', 'Compress', 'Decompress', 'Verify']}>
<Tabs.Tab>
```bash
# Download prebuilt binary (Linux x86_64, static musl)
wget https://github.com/LessUp/fq-compressor/releases/download/v0.2.0/\
  fq-compressor-v0.2.0-linux-x86_64-musl.tar.gz
tar -xzf fq-compressor-v0.2.0-linux-x86_64-musl.tar.gz
sudo mv fq-compressor-v0.2.0-linux-x86_64-musl/fqc /usr/local/bin/

fqc --version
```
</Tabs.Tab>

<Tabs.Tab>
```bash
# Compress a FASTQ file
fqc compress -i reads.fastq -o reads.fqc

# With threads and progress
fqc compress -i reads.fastq -o reads.fqc --threads 8
```
</Tabs.Tab>

<Tabs.Tab>
```bash
# Full decompression
fqc decompress -i reads.fqc -o restored.fastq

# Random access (range)
fqc decompress -i reads.fqc --range 1000:2000 -o subset.fastq

# Streaming (no temp files)
fqc decompress -i reads.fqc --stdout | gzip > output.fq.gz
```
</Tabs.Tab>

<Tabs.Tab>
```bash
# Verify archive integrity
fqc verify reads.fqc

# Show compression details
fqc info reads.fqc

# JSON output for scripting
fqc info reads.fqc --json | jq '.ratio'
```
</Tabs.Tab>
</Tabs>

## Technical Highlights

- **Compression Algorithms**: Assembly-based Compression (ABC) + Statistical Context Mixing (SCM)
- **Parallelism**: Intel oneTBB pipeline, tunable threads, zero-copy where possible
- **Format**: Custom FQC binary format with block-based structure for O(1) random access
- **Quality Modes**: Lossless, lossy, or discard quality scores
- **Integrity**: xxh64 checksums, built-in verification
- **Transparency**: Read directly from .gz, .bz2, .xz FASTQ inputs
