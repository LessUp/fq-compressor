# Performance Benchmarks

## Quick Results

### Test Setup

- **Hardware**: Intel Core i7-9700 @ 3.00GHz (8 cores)
- **Dataset**: 2.27M Illumina reads (511 MB uncompressed)
- **Build**: Release mode with LTO

### Compiler Comparison

| Compiler | Compression | Decompression | Ratio |
|----------|-------------|---------------|-------|
| GCC 15.2 | 11.30 MB/s | 60.10 MB/s | 3.97× |
| Clang 21 | 11.90 MB/s | 62.30 MB/s | 4.97× |

### Single vs Multi-threaded

| Threads | Compression | Decompression | Scaling |
|---------|-------------|---------------|---------|
| 1 | `3.2` MB/s | `15` MB/s | `1.0×` |
| 4 | `9.8` MB/s | `45` MB/s | `3.0×` |
| 8 | `11.9` MB/s | `62` MB/s | `3.7×` |

## Tool Comparison

| Tool | Ratio | Compress | Decompress | Parallel | Random Access |
|------|-------|----------|------------|----------|---------------|
| **fq-compressor** | 3.97× | 11.9 MB/s | 62 MB/s | ✓ | ✓ |
| Spring | 4.5× | 3 MB/s | 25 MB/s | ✓ | ✗ |
| fqzcomp5 | 3.5× | 8 MB/s | 35 MB/s | ✗ | ✗ |
| gzip | 2.1× | 45 MB/s | 180 MB/s | ✗ | ✗ |
| pigz | 2.1× | 150 MB/s | 180 MB/s | ✓ | ✗ |
| zstd | 2.5× | 220 MB/s | 500 MB/s | ✗ | ✗ |

## Running Benchmarks

### Quick Benchmark

```bash
./scripts/benchmark.sh --quick --input data.fastq
```

### Full Comparison

```bash
# Build both compilers
./scripts/build.sh gcc-release
./scripts/build.sh clang-release

# Run comparison
python3 benchmark/compiler_benchmark.py \
    --input test_data.fastq.gz \
    --gcc-binary build/gcc-release/src/fqc \
    --clang-binary build/clang-release/src/fqc \
    --threads 1 4 8 \
    --visualize
```

### Multi-Tool Comparison

```bash
python3 benchmark/benchmark.py \
    --input data.fastq \
    --tools fqc,gzip,pigz,zstd \
    --threads 1,4,8
```

## Interpreting Results

### Compression Ratio

- **3.5-4×**: Typical for genomic data
- **`>4×`**: High redundancy data
- **`<3.5×`**: Noisy or diverse data

### Speed

- **Compression**: I/O bound + CPU bound
- **Decompression**: Primarily I/O bound
- **Multi-threading**: Diminishing returns after 8 cores

### Trade-offs

```
Max Compression    Balanced    Max Speed
     ↓                ↓            ↓
  Spring      fq-compressor    pigz
  (4.5×)         (4.0×)      (2.1×)
```

## Continuous Benchmarking

### CI Integration

```yaml
- name: Benchmark
  run: |
    python3 benchmark/compiler_benchmark.py \
        --input benchmarks/test_data.fq.gz \
        --gcc-binary build/gcc-release/src/fqc \
        --clang-binary build/clang-release/src/fqc \
        --output-dir benchmark-results/

- name: Upload Results
  uses: actions/upload-artifact@v4
  with:
    name: benchmark-results
    path: benchmark-results/
```
