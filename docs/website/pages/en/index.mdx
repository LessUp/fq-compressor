# fq-compressor

High-performance FASTQ compression with 3.97× ratio and O(1) random access.

```bash
# Install
wget https://github.com/LessUp/fq-compressor/releases/download/v0.2.0/fq-compressor-v0.2.0-linux-x86_64-musl.tar.gz
tar -xzf fq-compressor-v0.2.0-linux-x86_64-musl.tar.gz
sudo mv fq-compressor-v0.2.0-linux-x86_64-musl/fqc /usr/local/bin/

# Compress
fqc compress -i reads.fastq -o reads.fqc

# Decompress
fqc decompress -i reads.fqc -o output.fastq

# Random access
fqc decompress -i reads.fqc --range 1000:2000 -o subset.fastq
```

## Performance

| Metric | Value |
|--------|-------|
| Compression ratio | 3.97× (Illumina) |
| Compression speed | 11.9 MB/s |
| Decompression speed | 62.3 MB/s |
| Random access | O(1) |

## Features

- **ABC + SCM hybrid compression** — Near entropy limit
- **O(1) random access** — Block-indexed, no full decompression needed
- **Transparent input** — Direct .gz/.bz2/.xz support
- **Long read support** — Adaptive PacBio/Nanopore handling
- **Data integrity** — xxh64 checksum verification
