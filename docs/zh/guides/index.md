# 指南

指南部分面向实际流程：构建项目、运行命令、验证输出，以及在不脱离白皮书语境的前提下参与贡献。

## 索引

- [安装](./installation)：在 Linux 发布二进制和本地源码构建之间做选择。
- [快速开始](./getting-started)：安装完成后，做一次可验证的烟雾级往返。
- [CLI 使用](./cli-usage)：`compress`、`decompress`、`info`、`verify` 的紧凑命令面。
- [贡献](./contributing)：closeout 模式下的工作流、质量门槛与文档要求。

## 如何使用本节

当你已经接受项目定位、只需要一条可执行路径时，就从这里开始。如果某个选项或流程需要系统上下文，请同时参考 [架构](/zh/architecture/)。
